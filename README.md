## 🚀 What It Does

InvoiceFlow is a fully client-side invoice generator. Fill in your details, add line items, and download a professional PDF — no backend, no account, no install required. Works on mobile and desktop.

**Features:**

- Add your business details and client details
- Multiple line items with quantity and rate
- Automatic subtotal, tax, and discount calculations
- Multiple currency support ($, €, £, ₦, ¥, A$, C$)
- Custom tax rates and discount percentages
- Notes and payment terms fields
- Live preview before downloading
- One-click PDF download via browser print

## ⚙️ Getting Started

```bash
cd frontend
npm install
npm run dev
```

Open `http://localhost:5173` — works instantly, no backend needed.
